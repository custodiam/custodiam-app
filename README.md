# Custodiam App

App móvil y web para gestión de agrupaciones de Protección Civil.

## Stack

- Flutter 3.x
- Dart 3.x

## Desarrollo

```bash
# Instalar dependencias
flutter pub get

# Ejecutar en desarrollo
flutter run

# Ejecutar en Chrome
flutter run -d chrome

# Build web
flutter build web

# Tests
flutter test

# Análisis estático
flutter analyze
```
